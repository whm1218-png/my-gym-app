# 6주 스프린트 일정 및 WBS 3단계 매핑

학기 단위 **`05-wbs-schedule.md` (15주)** 가 전체 시간축이라면, 본 문서는 **실행 가능한 MVP 압축 일정**(6개 스프린트)이다. 학기에서는 **연속 두 번**(예: 개발 초기 6주 + 개발 후기 6주)으로 반복 매핑해도 된다.

**대응 관계 요약**

|------|---------------------|
| 1주 | 약 1~2주 분량 압축 |
| 2주 | 약 3주 분량 |
| 3주 | 약 4~5주 분량 |
| 4주 | 약 6주 분량 |
